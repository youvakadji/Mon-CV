Etnaudit - Logiciel d'audit de sécurité (équipe de 5)

Objectif du projet :

Etnaudit est un logiciel conçu pour effectuer des audits de sécurité destinés aux particuliers, développé en équipe de 5 personnes. Ce projet intègre plusieurs technologies et outils modernes pour offrir une expérience complète et sécurisée.

Technologies :

Django REST Framework - Python - Pytest - ReactJS - TypeScript - ElectronJS - Postman - Mysql - GitHub Actions - déploiement sur Heroku
