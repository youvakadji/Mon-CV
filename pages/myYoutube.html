MyYouTube - Plateforme de vidéos (équipe de 2)

Objectif du projet :

MyYouTube est une plateforme inspirée de YouTube, développée en microservices pour répondre aux défis de scalabilité et d'architecture moderne. Réalisé en duo, ce projet met en avant une utilisation avancée des technologies web et mobiles.

Technologies :

VueJS - TypeScript - NodeJS - ExpressJS - Prisma - React Native - Docker - Postman - Mysql - Elasticsearch - Postfix
